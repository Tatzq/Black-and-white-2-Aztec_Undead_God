Need to install Black-White-2-Reux 1.8

Shortcut keys:

SHIFT+S Death Miracle

SHIFT+D Decay Miracle

SHIFT+T Temple Death Miracle （The created undead cavalry has a time limit）

In your city center, range 8, click to activate the Undead God City Center Spell. Discard the dead villagers to the town center, and the villagers will be resurrected. The resurrected villagers are skeletons and cannot become believers.

Shift+8 can be used to modify player colors（The functionalities provided by the Tribal Pack DLL）
